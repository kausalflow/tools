How to use the data

There three different ways to use the data.

  1. Use the website, tools.kausalflow.com.
  2. Use RSS feed, RSS Feed. Use feed reader to follow new contents.
  3. Use the JSON data, JSON Data. You can programmatically access the data using the programming language of your choice. Here is an example using Python. This JSON data also makes it easy for you to create other application based on this data, e.g., chrome new table showing random tools.
