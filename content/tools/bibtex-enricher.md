summary: "BibTeX Enricher is a free, browser-based tool that automatically enriches .bib reference files with abstracts and missing metadata by querying multiple academic databases in batch. It requires no installation and runs entirely in the browser, making it instantly accessible to any researcher working with BibTeX bibliographies."
features:
  - Batch queries CrossRef, OpenAlex, Semantic Scholar, Europe PMC and Unpaywall in parallel
  - AI-powered web search fallback (via Claude) for entries still missing abstracts after database queries
  - Reconstructs OpenAlex abstracts from inverted index format
  - Merges metadata intelligently — longest abstract wins, never overwrites existing fields
  - Enriches keywords from MeSH terms, OpenAlex concepts and Semantic Scholar fields of study
  - Adds citation counts, open access status, PMID, PMCID, ISSN and OA PDF links
  - Live per-entry status indicators showing which sources returned data
  - Download enriched .bib file directly from the browser
